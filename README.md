# Mis Proyectos - Ingeniería de Computación (USB)

Este repositorio contiene los laboratorios y proyectos desarrollados para el curso Algoritmos 3.

## 📁 Estructura del Repositorio

* **Lab1:** Implementación del algoritmo de **Degrees of Separation** utilizando Kotlin. Se enfoca en el manejo de grafos y búsqueda de caminos más cortos.
* **Lab2:** Implementación de un solucionador de Sudoku utilizando Backtracking en Kotlin. Se enfoca en la búsqueda recursiva con condiciones de poda (validación de filas, columnas y subcuadrículas) para encontrar soluciones de forma eficiente.
